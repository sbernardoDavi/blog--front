## Getting Started

This project includes the creation of a website for the law course of the University of Valença (UNIFAA) in Rio de Janeiro.

The website will be built using Next.js, a popular React framework for building server-side rendered applications. The website will include features such as a homepage, course information, and a contact page.

### Installation

To get started, clone the repository and install the dependencies:

```bash
git clone
cd unifaa-law-course-website
npm install
```

### Running the Development Server

To start the development server, run:

```bash
npm run dev

```

This will start the server at `http://localhost:3000`, where you can view the website in your browser.

### Building for Production

To build the application for production, run:

```bash
npm run build
```

This will create an optimized production build in the `.next` directory. You can then start the
production server with:

```bash
npm start
```

This will start the server at `http://localhost:3000`, where you can view the
production version of the website.
